# Useful Libraries
***

## Python
> OpenCV: Computer Vision  
> Numpy: Matrix Calculation Library similar to MATLAB  
> SciPy: Numerical integration, optimization, and signal filtering  
> Matplotlib / Seaborn: Data visualization  
> PyTorch / TensorFlow: Machine learning and deep learning  
> Pandas: Data logging and analysis

## ROS 2
> rclpy: ROS2 Python Client Library  
> usb_cam: A ROS Driver for V4L USB Webcam  
> tf2: Coordinate transforms between frames  
> Navigation2 (nav2): Path planning and autonomous navigation  
> ros2_control: Thruster and actuator control interface standardization. Useful when working in simulation and hardware. <br>
> cv_bridge: Interface between ROS images and OpenCV  
> micro-ROS: ROS 2 for embedded microcontrollers
